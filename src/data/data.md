# Features description
### mag: 
The magnitude of the earthquake.
*Moc trzęsienia ziemi.*

### depth: 
The depth of the earthquake in kilometers.
*Głębokość trzęsienia ziemi w kilometrach.*

### nst: 
The number of seismic stations that reported P- and S-arrival times for this earthquake.
*Liczba stacji sejsmicznych, które zarejestrowały czasy przybycia fal P i S dla tego trzęsienia ziemi.*

### gap: 
The largest azimuthal gap between azimuthally adjacent stations (in degrees).
*Największa luka azymutalna między sąsiadującymi azymutowo stacjami (w stopniach).*
